# Simulated-E-Banking-System
## Overview
The simulated e-banking system is implemented as a web-based banking platform. The system allows customers, bankers and administrators to perform standard banking operations. The backend is fully mocked along with the data. No actual financial services are used.
## Features
- Secure login with multi-factor authentication
- View account balances and transaction history
- Customer tools to transfer funds between accounts and pay bills
- Banker tools for managing accounts and approving transactions
- Admin tools for monitoring system activity and generating reports

### How to Run Locally
1. Clone the repository
2. Install Dependencies \

3. Start the Server \
{ node Backend/server.js }
4. Open in browser
